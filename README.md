# Kostyle4U

다양한 기능을 추가하여 흥미를 유발하는 사용자 맞춤형 웹 솔루션 Kostyle4U (의류 쇼핑몰 웹 솔루션)

## 프로젝트 소개

<p align="justify">
SNS 활성화 시대에 맞춰 패션의 다양성을 타인과 공유할 수 있고, 기존 의류 사이트의 난잡한 UI와 한정적인 서비스의 한계성을 보완해 사용자들의 흥미를 유발하는 기능 및 개인 맞춤 추천 서비스를 추가한 의류 쇼핑몰 개발 프로젝트입니다.
</p>
